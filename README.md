# personal-site
Code to my personal site, hosted on AWS S3

## Getting Started
To run the website in development, execute the following:
```bash
git clone git@github.com:jack-bischoff/personal-site.git
npm install
npm start
```
The app will start a webserver at
```
localhost:8080
```
